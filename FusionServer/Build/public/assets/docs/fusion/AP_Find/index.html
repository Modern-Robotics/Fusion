Find/Connect MyBot Access Point

Find default MyBot Access Point

Each MyBot has its own unique access point Service Set Identifier (SSID) assigned to it. The access point starts with RecruitAP_ followed by a 6 digit alphanumeric value unique to each controller. Therefore a valid MyBot Access Point is RecruitAP_99be01.

This access point can be connected to via Wi-Fi from a device capable of Wi-Fi and browser access (i.e. computer). Using the device's Wi-Fi capabilities, find the MyBot Access Point and connect to it.

  • Connect via iOS
  • Connect via Android

Connect to the default MyBot Access Point

Select the MyBot Access Point that matches the SSID on the bottom of your controller (also inside box) to the SSID found.
The default MyBot passkey is mrirecruit. Once connected, open your web browser (Google Chrome recommended) and follow the steps to access to the web interface.

For instructions on changing the MyBot Access Point SSID and Passkey, refer to Change MyBot Access Point instructions.
