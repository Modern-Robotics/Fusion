Intermediate Display Robot Control

The Display toolbox contains blocks to show emojis, colors, and text on the display. There is also an additional block to clear the display.

List of available blocks:

  • Display Color
  • Display Emoji
  • Display Face
  • Display Text
  • Clear Display

Display Color

Shows the selected color on the display.

Block:

Code Produced:

import Display

Display.show_color("red")

Display Emoji

Shows the selected emoji on the display.

Block:

Code Produced:

import Display

Display.show_emoji("eyes_straight")

Display Face

Shows the selected face on the display.

Block:

Code Produced:

import Display

Display.show_face("straight")

Display Text

Shows the text value on the display.

Block:

Code Produced:

import Display

Display.show_text("Hello World")

Clear Display

Clears the display by showing a clear white background.

Block:

Code Produced:

import Display

Display.show_color("white")
